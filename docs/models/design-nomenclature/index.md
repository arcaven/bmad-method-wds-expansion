# Design Nomenclature Reference

A comprehensive vocabulary for describing visual design styles, enabling clear communication between designers, developers, and AI tools.

---

## Purpose

This reference helps you:
- **Communicate design intent** precisely with teams and stakeholders
- **Prompt AI tools** (Midjourney, DALL-E, code generation) effectively
- **Document Visual Direction** in WDS projects
- **Understand design feedback** from clients and collaborators

## Using This Reference

### In Visual Direction Documents

When documenting visual direction for a project, use specific terminology:

```markdown
## Visual Style
- Primary: Neobrutalism with warm palette
- Typography: Geometric sans-serif (display), Humanist sans (body)
- Layout: Card-based with asymmetrical balance
- Effects: Soft shadows, subtle grain texture
```

### For AI Prompting

Combine terms for precise results:

```
"Create a glassmorphism card component with
duotone icons, geometric sans-serif typography,
and subtle drop shadows"
```
